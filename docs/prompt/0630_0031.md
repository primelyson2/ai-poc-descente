4. before/after buffer, elapsed, disk 등을 비교해서 개선이 되었는 지 확인함.
5. Vector 조회를 해서 비슷했던 결과서의 내용을 요약해서 결과서에 넣는다. (링크 전달도 방법)
6. 결과서를 기존과 동일하게 작성한다.
7. 결과서를 백터로 저장한다.
